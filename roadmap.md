# VI. Roadmap

## Fase 1 — Fundación (Q3-Q4 2026)

* Smart contracts auditados y desplegados en Polygon (mainnet)
* MVP del Pressing Studio (upload + análisis + creación)
* Marketplace básico con ventas primarias
* Player de reproducción
* Onboarding de 10-20 artistas pioneros de renombre

---

## Fase 2 — Expansión (Q1-Q2 2027)

* Mercado secundario con royalties automáticos
* Soporte multi-chain (Avalanche)
* Bound Content completo (podcasts, video, acceso a eventos)
* App móvil (iOS + Android)
* Herramientas para sellos discográficos y programa de embajadores

---

## Fase 3 — Ecosistema (Q3 2027+)

* Integración de catálogos de grandes sellos discográficos
* Gobernanza comunitaria (votación por holders de Pressings, sin token especulativo)
* API abierta para integraciones de terceros
* Expansión a arte visual, literario y audiovisual
* Partnerships con venues y festivales
* Programa de curadores
